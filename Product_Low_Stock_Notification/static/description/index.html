Product Low Stock Notification

- Under settings>User&Companies> User To get low-stock product notifications users have to enable "Notify User". (Photo 1)

- Under Settings > User & Companies > Companies menu user has enabled "Low Stock Notification?". (Photo 2)

- To get the notification for global product on hand or forecast minimum quantity user has to select "Min Quantity Based On - Global for all products" and "Apply On - Product". (Photo3)

- The user can put his data as a sender in Gmail or Outlook, and then we will test the connection to confirm the validity of the data. (Photo 4)

- Users can set corn for the email-sending function. (Photo 5)

- Users can see product minimum quantity low stock notification email reminders to notify the user. (Photo 6)

- Users can see the product variant minimum quantity low stock report attached to the email. (Photo 7)

- To get the notification for global product on hand or forecast minimum quantity user has to select "Min Quantity Based On - Global for all products" and "Apply On - Product". (Photo 8)

- On the product form view user has to set a minimum quantity for the specific product. (Photo 9)

- Users can see the product minimum quantity low stock report attached to the email. (Photo 10)

- To get the notification for a product on hand or forecast minimum quantity based on reorder rules users have to select "Min Quantity Based On - Reorder Rules" and "Apply On - Product". (Photo 11)

- Users have to create reordering rules for products and set a "Minimum Quantity" for products. (Photo 12)

- Users can see the product minimum quantity low stock report attached to the email. (Photo 13)
